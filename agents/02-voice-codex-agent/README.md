## Brick Types Explained

### üí° Insight
**What**: Realization or discovery  
**Example**: *"I realized my clients don't fail because they lack tactics - they fail because they lack a decision-making framework."*  
**When to use**: Original observations, "aha moments"

### üé® Metaphor
**What**: Comparison that illuminates  
**Example**: *"Your personal brand is an appreciating asset, not a depreciating expense."*  
**When to use**: Visual explanations, memorable comparisons

### ‚ö° Distinction
**What**: Differentiation that creates clarity  
**Example**: *"Prompting tools is like tweeting. Agent building is like founding."*  
**When to use**: Contrasting concepts, clarifying differences

### üîÅ Refrain
**What**: Repeated phrase or belief  
**Example**: *"AI can scale your voice, but only you can make that voice worth scaling."*  
**When to use**: Signature phrases, core beliefs

### üìê Framework
**What**: Structured thinking model  
**Example**: *"The MVP Framework: Not Minimum Viable Product, but Minimum Viable Personality."*  
**When to use**: Step-by-step approaches, methodologies

### ‚ùì Question
**What**: Powerful reframing question  
**Example**: *"What would this look like if it were easy?"*  
**When to use**: Perspective shifts, diagnostic questions

## Quality Scores

Each brick rated on 4 dimensions (1-10):

**Novelty**: How original is this?
- 8-10: Rarely expressed this way
- 4-7: Somewhat unique
- 1-3: Common knowledge

**Utility**: How useful is this?
- 8-10: Broadly applicable
- 4-7: Useful in specific contexts
- 1-3: Interesting but not actionable

**Clarity**: How clear is this?
- 8-10: Crystal clear standalone
- 4-7: Clear with context
- 1-3: Vague or confusing

**Memorability**: How sticky is this?
- 8-10: Quotable, repeatable
- 4-7: Somewhat memorable
- 1-3: Forgettable phrasing

**Overall** = Average of four scores

## Best Practices

### ‚úÖ DO
- Capture thoughts immediately (don't wait)
- Speak/write naturally (not formally)
- Add context when available
- Review bricks weekly
- Connect related ideas
- Export regularly

### ‚ùå DON'T
- Edit while capturing (preserve raw thinking)
- Force formal structure
- Delete "bad" bricks (low scores still useful)
- Capture others' ideas without attribution
- Skip tagging

## Tips for Great Bricks

1. **Record in the moment** - Fresh insights are clearest
2. **Think out loud** - Don't polish, just speak
3. **Note the trigger** - What sparked this thought?
4. **Keep it atomic** - One idea per brick
5. **Make it memorable** - Test if you can recall it

## Troubleshooting

**Microphone not working?**
- Check browser permissions
- Use text input instead
- Try different browser

**Bricks not extracting well?**
- Be more explicit in language
- Use brick type indicators (see prompt)
- Add more context

**Can't find brick later?**
- Use search function
- Add more tags
- Review themes
