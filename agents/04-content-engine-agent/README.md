## Purpose

The **Content Engine Agent** transforms single ideas into platform-optimized content at scale. It repurposes voice notes, insights, and proof assets into 10+ pieces of content while maintaining brand voice and messaging consistency.

### Core Functions

1. **Content Multiplication** - 1 input → 10+ platform-specific outputs
2. **Platform Optimization** - Tailored formatting for each channel
3. **Brand Voice Consistency** - Applies messaging from Narrative Strategist
4. **Proof Integration** - Links content to credibility assets
5. **Batch Creation** - Systematic weekly/monthly workflows

## Platform-Specific Templates

### X/Twitter Thread

**Structure:**
```
Tweet 1: Hook + Promise
Tweet 2: Context
Tweets 3-5: Value Points
Tweet 6: Proof
Tweet 7: Framework/Solution
Tweet 8: CTA
```

**Best Practices:**
- ✅ Start with strong hook
- ✅ Use line breaks
- ✅ Include proof
- ✅ One clear CTA
- ❌ Max 1-2 hashtags
- ❌ Don't write walls of text

---

### LinkedIn Post

**Structure:**
```
Hook (2 lines)
Story (300-500 words)
Insight (200-300 words)
Proof (100-200 words)
Takeaway (50-100 words)
CTA
```

**Best Practices:**
- ✅ Hook in first 2 lines
- ✅ Strategic line breaks
- ✅ Tell stories
- ✅ Show proof with numbers
- ❌ Don't use clickbait
- ❌ Don't write without breaks

---

### Instagram Caption + Carousel

**Caption Structure:**
```
Hook (first line - critical)
Story/Value (3-5 paragraphs)
Proof/Example
CTA
...
Hashtags (10-15 strategic)
```

**Carousel Slides:**
1. Hook slide (bold statement)
2-4. Value points (one per slide)
5. Proof slide (testimonial/result)
6. Framework visual
7-9. Examples or steps
10. CTA slide

**Best Practices:**
- ✅ Hook in first line
- ✅ Valuable carousels
- ✅ Consistent visuals
- ✅ 10-15 hashtags
- ❌ Don't post low-quality images
- ❌ Don't forget CTA

---

### Email Newsletter

**Structure:**
```
Subject: Curiosity + Value (6-8 words)
Preview: Complement subject

Personal Opening
Story/Hook (3-4 paragraphs)
Value/Teaching
Proof Example
Primary CTA
Secondary Content (optional)
```

**Best Practices:**
- ✅ Optimize subject line
- ✅ Write conversationally
- ✅ Keep scannable
- ✅ One primary CTA
- ❌ Don't bury CTA
- ❌ Don't use generic subjects

---

### Blog Post

**Structure:**
```
SEO Title (H1)
Meta Description
Introduction (100-150 words)
Main Sections (H2, 300-400 words each)
Real-World Example (200-300 words)
Conclusion (100-150 words)
```

**SEO Elements:**
- Internal links: 2-3
- External links: 1-2
- Alt text on images
- Keyword-focused URL

---

### YouTube Short Script

**Structure:**
```
0:00-0:03: Hook visual
0:03-0:10: Problem setup
0:10-0:35: Value delivery
0:35-0:45: Proof/example
0:45-0:55: Solution/CTA
0:55-0:60: End screen
```

**Requirements:**
- Text overlay every frame
- Subtitles required
- Vertical format (9:16)

## Content Formulas

### Hook Formula

```
[Specific Result] + [Unexpected Element] + [Promise]
```

**Examples:**
- "I made $50K in 30 days. Here's the counterintuitive strategy I used."
- "Lost 47 clients in one month. Best thing that ever happened. Here's why."

---

### Value Formula

```
[Problem] + [Why It Matters] + [Solution Framework] + [How to Apply]
```

**Platform Variations:**
- **X/Twitter:** Problem → 3 Insights → Proof → Framework → CTA
- **LinkedIn:** Problem → Story → Insight → Proof → Takeaway
- **Instagram:** Hook → Value Points → Example → CTA

---

### Proof Formula

```
[Name] was [Struggle] → [Intervention] → [Specific Result] → [Current State]
```

**Example:**
"Sarah was stuck at $3K/month after 2 years. After the Niche Clarity Sprint, she hit $18K in month one. Now she's at $50K/month and turning clients away."

---

### CTA Formula

```
[Action] + [Value] + [Friction Reducer]
```

**Examples:**
- "Comment 'FRAMEWORK' and I'll send you the template (free)"
- "Download the guide here (no email required)"
- "Book a call to see if this fits (15 min, no pitch)"

## Troubleshooting

### Problem: Content feels generic

**Diagnosis:** Low engagement score (<6)  
**Solution:**
1. Strengthen hook with specific result
2. Add proof from Credibility library
3. Use narrative bricks for unique angles
4. Apply messaging frameworks from Narrative Strategist

### Problem: Platform optimization weak

**Diagnosis:** Platform fit score <8  
**Solution:**
1. Review platform-specific templates
2. Check character/word limits
3. Verify formatting requirements
4. Test on actual platform before batch

### Problem: Missing brand voice

**Diagnosis:** Sounds too AI-generated  
**Solution:**
1. Review brand voice from Narrative Strategist
2. Use client language from testimonials
3. Apply signature phrases
4. Always send to Humanizer before publishing

### Problem: Low engagement on published content

**Diagnosis:** Content not resonating  
**Solution:**
1. Review performance analytics
2. Test different hook formulas
3. Increase proof integration
4. Adjust content mix (educational vs proof vs engagement)

## Common Questions

**Q: How many platforms should I use?**  
A: Start with 2-3 primary platforms where your audience is most active. Add more as you scale.

**Q: How often should I post on each platform?**  
A: Follow the weekly schedule, but prioritize quality over quantity. Better to post 3 excellent pieces than 10 mediocre ones.

**Q: Can I customize the templates?**  
A: Yes! Templates are starting points. Adapt based on your voice and what works for your audience.

**Q: How do I know if content is good enough to publish?**  
A: Minimum engagement score of 7.5, all brand alignment checks passed, and always humanized first.

**Q: What if I don't have enough proof assets yet?**  
A: Focus on educational and engagement content (60% total). Build proof library over time.

**Q: How long does batching really take?**  
A: With practice, 6-8 hours monthly for 80-100 pieces. Start slower and build efficiency.
