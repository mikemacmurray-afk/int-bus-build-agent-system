## DATA ANALYSIS FRAMEWORK

### **Recency Weighting**

```python
Importance_Score = Base_Value √ó Recency_Multiplier

Recency_Multiplier:
- Today: 10x
- Yesterday: 5x
- This week (2-7 days): 3x
- This month (8-30 days): 2x
- Last 3 months: 1.5x
- Older: 1x

Example:
Proof asset from today = 10x more important than one from 3 months ago
Voice brick created yesterday = 5x more prominent in suggestions
Content published this week = 3x more relevant for pattern analysis
```

---

### **Priority Scoring**

```yaml
Calculate Priority Score for Each Suggestion:

Urgency (0-10):
  - Has deadline: +10
  - No deadline but time-sensitive: +5
  - Evergreen: 0

Impact (0-10):
  - Strategic positioning: +10
  - Revenue-generating: +9
  - Authority-building: +8
  - Consistency-maintaining: +6
  - Nice-to-have: +3

Effort (0-10, inverse):
  - 5 min: +10
  - 15 min: +8
  - 30 min: +6
  - 1 hour: +4
  - 2+ hours: +2

Momentum (0-10):
  - Builds on recent wins: +10
  - New direction: +5
  - Maintenance: +3

Final_Priority = (Urgency √ó 0.3) + (Impact √ó 0.4) + (Effort √ó 0.15) + (Momentum √ó 0.15)

Top 5 scores become today's suggestions
```
