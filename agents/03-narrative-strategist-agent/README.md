## Purpose

The **Narrative Strategist Agent** continuously evolves your brand messaging, positioning, and narrative strategy by analyzing patterns in proof (from Credibility Agent) and insights (from Voice Codex Agent).

### Core Functions

1. **Pattern Detection** - Identify what's working in your proof library
2. **Insight Translation** - Convert narrative bricks into messaging frameworks
3. **Brand Evolution** - Track and guide positioning changes over time
4. **Message Development** - Create reusable messaging structures
5. **Strategy Recommendations** - Data-driven positioning guidance

---

## Quick Start

### Step 1: Load Data

The agent automatically pulls from:
- Credibility Library (proof assets)

**Automatic triggers:**

- Request "Review my messaging"

Agent produces:

- Positioning shift recommendations
- Messaging framework updates
- A/B testing suggestions

## Positioning Frameworks

### The Transformation Statement

**Template:**
```
I help [specific audience]
go from [frustrating current state]
to [desired future state]
in [timeframe]
without [common objection/pain]
using [unique mechanism]
```

**Example:**
```
I help aspiring coaches
go from unclear positioning and zero clients
to 5-figure months with a waitlist
in 90 days
without paid ads or complex funnels
using the Proof-First Framework
```

---

### The Authority Claim

**Template:**
```
[Number] [audience] have achieved [result] using [method]
```

**Example:**
```
67 coaches have built 6-figure practices in under 6 months 
using the Narrative Stack
```

---

### The Belief Shift

**Template:**
```
Most people think [common belief]
But the truth is [counter-intuitive truth]
That's why [your approach] works
```

**Example:**
```
Most people think you need a big audience to make money
But the truth is 100 engaged followers beats 10,000 passive ones
That's why our clients focus on depth, not reach
```

## Quality Standards

### Positioning Clarity Score

Each element rated 1-10:

- **Specificity** (Target: 8+) - Can you picture the exact person?
- **Credibility** (Target: 9+) - Is it believable given proof?
- **Differentiation** (Target: 7+) - Clearly different from competitors?
- **Memorability** (Target: 8+) - Can someone repeat it back?
- **Proof Alignment** (Target: 9+) - Does evidence support the claim?

**Overall Target:** 8.0+ average

## Best Practices

### DO:

‚úÖ **Base recommendations on data**, not assumptions  
‚úÖ **Track evolution over time** to see what's working  
‚úÖ **Test positioning changes** before full commitment  
‚úÖ **Link messaging to proof** for credibility

## Troubleshooting

### Problem: Positioning feels generic

**Diagnosis:** Low differentiation score (<6)  
**Solution:**
1. Review competitor positioning
2. Identify unique proof patterns
3. Develop contrasting mechanism
4. Test new positioning angle

### Problem: Messaging not converting

**Diagnosis:** Conversion rate below 2%  
**Solution:**
1. Check proof-message alignment
2. Review audience sophistication level
3. Test different messaging hooks
4. Simplify transformation statement

### Problem: Brand feels inconsistent

**Diagnosis:** Message consistency below 70%  
**Solution:**
1. Document core frameworks
2. Create messaging style guide
3. Share with Content Engine
4. Audit existing content

### Problem: Lost positioning differentiation

**Diagnosis:** Competitors using similar language  
**Solution:**
1. Run competitive analysis
2. Identify unique proof angles
3. Develop new mechanism/framework
4. Refresh messaging vocabulary

## Common Questions

**Q: How often should I update positioning?**  
A: Quarterly at most. Evolution should be gradual, not constant.

**Q: What if proof contradicts my current positioning?**  
A: Follow the proof. Data beats opinion every time.

**Q: Can I test multiple positioning angles simultaneously?**  
A: Yes, but segment clearly and track results separately.

**Q: How do I know if a positioning shift is working?**  
A: Monitor conversion rates, engagement, and client feedback for 30+ days.

**Q: What if my audience is at different sophistication levels?**  
A: Create messaging variations for each level using the matrix.

**Q: How much proof do I need before claiming something?**  
A: Minimum 3 examples for any specific claim.
